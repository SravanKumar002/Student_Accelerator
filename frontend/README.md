# Student Accelerator

Coach dashboard and learning path tracker for students. Coaches can sign in, view student learning paths, and manage paths with clear visualizations.

## Tech stack

- Vite, TypeScript, React
- Tailwind CSS, shadcn/ui
- Supabase (auth + database)
- Recharts, Framer Motion

## Setup

1. Clone the repo and go into the project folder.
2. Install dependencies: `npm i`
3. Copy `.env.example` to `.env` and set your Supabase URL and anon key.
4. Run dev server: `npm run dev`

## Auth

Email/password and Google sign-in are supported. Configure Google OAuth in the Supabase dashboard (Authentication → Providers → Google) with your client ID and secret.

## Scripts

- `npm run dev` – start dev server
- `npm run build` – production build
- `npm run preview` – preview production build
- `npm run lint` – run ESLint
- `npm run test` – run tests
